## Usage

Setup virtualenv (optional)
```bash
virtualenv ve
. ve/bin/activate
```

Install dependencies
```bash
pip install -r requirements.txt
npm install
```

Run migrations
```bash
./manage.py migrate
```

Run django server
```bash
./manage.py runserver
```

Run webpack compiler
```bash
./node_modules/.bin/webpack --config webpack.config.js --watch
```
